User Registration - Test Case

Details
Steps & Expected Results
Attachments
Comments

This test case verifies that a new user can successfully register an account using valid data. The system should create a new user account, send a verification email, and redirect the user to the success page.

  1. Testing environment is up and running
  2. Database is accessible and configured properly
  3. Email service is configured and operational
  4. The email address used for registration does not already exist in the system

Test Steps

  1. Navigate to the registration page
  2. Enter valid data in all required fields:
    • First Name: John
    • Last Name: Doe
    • Email: john.doe@example.com
    • Password: StrongP@ss123
    • Confirm Password: StrongP@ss123
  3. Check the "I agree to Terms and Conditions" checkbox
  4. Click the "Register" button
  5. Verify that the success message is displayed
  6. Check the email inbox for a verification email
  7. Click on the verification link in the email
  8. Verify that the account is activated
  9. Try to log in with the created credentials

Expected Results

  1. Registration page loads without errors
  2. All fields accept the input without validation errors
  3. Checkbox can be selected
  4. Registration form submits successfully
  5. Success message "Your account has been created. Please check your email to verify your account." is displayed
  6. Verification email is received with a valid verification link
  7. Clicking the verification link redirects to the account verification page with a success message
  8. System shows the account as activated
  9. User can log in successfully with the created email and password
